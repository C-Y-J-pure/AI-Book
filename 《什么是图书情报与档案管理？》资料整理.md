
# 1912年
## 月不详
日不详，创设广东省立中山图书馆，服务于中山大学信息管理学院。
# 1927年
## 月不详
日不详，金陵大学设立图书馆学系。

# 1930 年至 1939 年
## 月不详
日不详，由民国政府在20世纪30年代发起的“行政效率运动”，将文书档案作为发布和执行政令的重要工具，一系列与之相关的论著出版问世。

# 1953年
## 9月
文华图专并入武汉大学，成立图书馆学系。

# 1979年
## 月不详
日不详，华东师范大学成立信息管理系。该系于1993年归入华东师范大学商学院。
# 1980年
## 月不详
日不详，中山大学创立了图书馆学大专班。
# 1982年
## 1月
2日，广州图书馆举行开馆典礼，广州图书馆为国家级本科实习教学基地建设单位。
## 月不详
日不详，国家教委发布的《普通高等学校社会本科专业目录》中，历史学学科门类下专门开设了“文博与档案专业”。


# 1983年
## 月不详
日不详，华中师范大学开设图书馆情报学专业，次年开始招收专科学生。

# 1984年
## 月不详
日不详，武汉大学经原国家教委批准成立图书馆情报学院。
# 1987年
## 4月
日不详，原湖南医科大学成立医学图书情报学系成立（现中南大学湘雅医学院医药信息系），同年秋季开始招生，后续1999 年至 2002 年期间变更为信息管理与信息系统专业（医学方向），学制五年，管理学门类。2003 年至今为医学信息学专业（全国首家），学制四年，属理学门类。

## 8月
日不详，云南大学历史系于1987年增设了图书馆学本科专业。

# 1988年
## 月不详


# 1993年
## 月不详
日不详，华中师范大学图书情报学专业更名为信息管理系。
日不详，云南大学档案学系又增设了情报学本科专业。

# 1998年
## 7月
6日，根据国家教育部发布的《普通高等学校本科专业目录和专业介绍》的规定，将情报学专业名称改为信息管理与信息系统专业。
## 月不详
日不详，档案学从历史学门类转到管理学门类，更多地与图书馆学、信息科学等学科建立联结，重点聚焦于对档案现象本身及档案工作的研究，从而形成脱离历史学的独有的知识体系。

# 20世纪末
日不详，我国设立“图书馆、情报与档案管理”这个一级学科。

# 2000年
## 月不详
日不详，武汉大学图书馆情报学院更名为信息管理学院。

# 2004年
## 3月
日不详，黑龙江大学成立黑龙江大学信息资源管理研究中心。

# 2006年
## 月不详
日不详，经国务院学位委员会批准，云南大学档案与信息管理系获得图书馆、情报与档案管理一级学科硕士学位授予权和档案学二级学科博士学位授予权。

# 2009年

# 2011年
## 3月
8日，国务院学位委员会办公室颁发了《学位授予和人才培养学科目录(2011年)》，将“图书馆、情报与档案管理”这个一级学科命名为“图书情报与档案管理”[中华人民共和国教育部](http://www.moe.gov.cn/srcsite/A22/moe_833/201103/t20110308_116439.html)

# 2012年

# 2016年

# 2018年
## 月不详
日不详，南京农业大学信息管理与信息系统专业为省级特色专业，并于2018年通过英国图书情报协会 CILIP 认证。

# 2019年
## 4月
19日，国家图书馆互联网信息战略保存项目在北京启动，首家互联网信息战略保存基地落户新浪。这个项目是为了联合国内各信息保存机构，在全国范围内构建分级分布的中文网络信息资源采集与保存体系，保存互联网时代的国家记忆和数字遗产而启动的。[国家图书馆启动互联网信息战略保存项目](https://www.mct.gov.cn/whzx/zsdw/zggjtsg/201904/t20190423_843029.htm)

29日，教育部、中央政法委和科技部等13个部门在天津联合启动“六卓越一拔尖”计划2.0。情报学作为图书情报与档案管理学科的一部分，与“六卓越一拔尖”计划2.0中的新文科建设有着紧密的联系。[[中国网]教育部启动“六卓越一拔尖”计划2.0 - 中华人民共和国教育部政府门户网站](http://www.moe.gov.cn/jyb_xwfb/xw_zt/moe_357/jyzt_2019n/2019_zt4/tjx/mtjj/201904/t20190430_380243.html)

## 月不详

日不详，云南大学档案学专业首批入围教育部“双万计划”国家级一流本科专业建设队列。
# 2021年
## 10月
